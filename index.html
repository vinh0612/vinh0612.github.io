1. Database Security in SQL Server

Công nghệ: SQL Server, T-SQL (Dự án cá nhân)

Mô phỏng hệ thống quản lý sinh viên an toàn. Áp dụng mã hóa mức cột (Column-Level Encryption) với thuật toán AES-256 để bảo vệ dữ liệu nhạy cảm. Triển khai phân quyền Role-Based Access Control (RBAC) kết hợp Stored Procedures Proxy để chặn truy cập trực tiếp vào Data Tables.
